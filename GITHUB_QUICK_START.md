## ❓ Troubleshooting

### Files not showing on GitHub

**Solution:** Refresh your browser (Ctrl+Shift+R)

### "Can't find file upload button"

**Solution:** Make sure you're logged in, then:
1. Go to your repo
2. Click green **Code** button
3. Click **Upload files** tab

### File uploaded to wrong location

**Solution:** Use GitHub web interface to move files or delete and re-upload

### Want to delete and re-upload

1. Go to **Settings** → **Danger Zone** → **Delete this repository**
2. Create a new one
3. Upload again
